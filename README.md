# Cognia

What is Cognia? 

Cognia is a Python library that automatically performs exploratory data analysis (EDA) on datasets and generates structured insights, visualizations, and summaries with minimal user effort.

Instead of manually writing repetitive code to understand a dataset, Cognia allows users to analyze data in a single line of code, saving time and reducing cognitive load.
